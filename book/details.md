# Philosophy
หนังสือเล่มนี้ออกแบบมาเพื่อแก้ปัญหาและปลดล็อกศักยภาพในตัวของผู้หญิง INFJ ที่มักขาดความมั่นใจ สื่อสารไม่เก่ง เข้าใจยาก หรือมักดึงดูดคนที่มีปัญหาเข้ามาในชีวิต เนื้อหาจะเน้นการใช้จุดเด่นจากกลไกการทำงานของ Cognitive Function ประยุกต์เข้ากับทฤษฎีศิลปะการดึงดูดใจ เพื่อสร้างกลยุทธ์เสน่ห์แบบ Passive Attractive ที่ช่วยให้ผู้หญิง INFJ สามารถนำเสนอตัวเองได้อย่างเป็นธรรมชาติ เป็นผู้รับอย่างภาคภูมิใจ และสามารถดึงดูดรวมถึงครองใจคนที่รักได้โดยไม่ต้องฝืนเป็นฝ่ายรุก 

# Objective
1. เพื่อให้รู้กลไกการสร้างเสน่ห์ของผู้หญิง infj แบบไม่ต้องเป็นฝ่ายรุก 
2. เพื่อให้รู้จักกลไกการทำงาน cognitive function ของตัวเอง 
3. เพื่อให้สามารถดึงดูดคนที่รักได้ 
4. เพื่อสร้างเสน่ห์ให้ในแบบผู้หญิง infj 
5. เพื่อสร้างผู้หญิง infj ให้มีกลยุทธิ์ในการสร้างเสน่ห์และครองใจคนรักได้ 
6. สอนการใช้ cognitive function ของ infj เพื่อเป็นจุดเด่นในการสร้างเสน่ห์ 

# Constrain
1. แบ่งเนื้อหาออกเป็นอย่างน้อย 3 ส่วน 
2. จำนวนเนื้อหามี 12 บท โดยแต่ละบทต้องมีหัวข้อย่อยไม่น้อยกว่า 3 หัวข้อย่อย 
3. เนื้อหาต้องมีการอ้างอิงทฤษฎีเรื่อง MBTI 
4. นำเนื้อหาและทฤษฎีจาก Art of Seduction เข้ามาร่วมในเนื้อหา 
5. มีรูปภาพแทรกในเนื้อหา โดยจำกัดบทละ 1 รูปเท่านั้น 
6. มีเนื้อหาเกี่ยวกับการจับคู่ระหว่างผู้หญิง INFJ กับผู้ชาย Type อื่นๆ 
7. รูปแบบการเขียนต้องเขียนให้เหมือนเป็นเจ้าของความรู้ (ผู้เชี่ยวชาญ) โดยไม่ต้องอ้างอิงชื่อหนังสือแหล่งที่มา

# Philosophy
หนังสือเล่มนี้เป็นภาคต่อที่พัฒนาเนื้อหาและกลยุทธ์เชิงลึกต่อเนื่องจาก Vol. 1 โดยจะพาดำดิ่งลงไปในกลไกของ Cognitive Function และศาสตร์แห่งการดึงดูดที่ซับซ้อนยิ่งขึ้น เพื่อให้ผู้หญิง INFJ มีความเชี่ยวชาญในการสานสัมพันธ์และรับมือกับความรักในระยะยาว หนังสือเล่มนี้ถูกออกแบบมาเพื่อผู้อ่านที่มีความเข้าใจในพื้นฐานของตัวเองอย่างถ่องแท้แล้ว และต้องการต่อยอดชั้นเชิงในการดึงดูดและจับคู่กับผู้ชาย Type ต่างๆ อย่างมีศิลปะ 

# Objective
1. เพื่อนำเสนอกลยุทธ์และต่อยอดความรู้เชิงลึกที่มีความต่อเนื่องจากเนื้อหาในเล่มแรก 
2. เพื่อให้ผู้หญิง INFJ เข้าใจพลวัตการจับคู่กับผู้ชาย Type อื่นในระดับที่ลึกซึ้งยิ่งขึ้น 
3. เพื่อสร้างความมั่นใจในการพัฒนาความสัมพันธ์ระยะยาวด้วยการประยุกต์ใช้ Cognitive Function และ Art of Seduction 

# Constrain
1. แบ่งเนื้อหาออกเป็นอย่างน้อย 3 ส่วน 
2. จำนวนเนื้อหามี 12 บท โดยแต่ละบทต้องมีหัวข้อย่อยไม่น้อยกว่า 3 หัวข้อย่อย 
3. เนื้อหาต้องมีความเกี่ยวเนื่อง (Coherence) กับ book1 แต่ห้ามมีหัวข้อซ้ำกัน 
4. เนื้อหาออกแบบในลักษณะ Pre-requisite คือผู้อ่านจำเป็นต้องอ่าน book1 มาก่อนจึงจะสามารถทำความเข้าใจเนื้อหาเล่มนี้ได้ 
5. เนื้อหาต้องมีการอ้างอิงทฤษฎีเรื่อง MBTI 
6. นำเนื้อหาและทฤษฎีจาก Art of Seduction เข้ามาร่วมในเนื้อหา 
7. มีรูปภาพแทรกในเนื้อหา โดยจำกัดบทละ 1 รูปเท่านั้น 
8. มีเนื้อหาเกี่ยวกับการจับคู่ระหว่างผู้หญิง INFJ กับผู้ชาย Type อื่นๆ
