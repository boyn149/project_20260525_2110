# Layer 1: Intent - The Introverted Siren: มนตร์เสน่ห์ฉบับผู้หญิง INFJ Vol. 1

**Philosophy**
หนังสือเล่มนี้ออกแบบมาเพื่อแก้ปัญหาและปลดล็อกศักยภาพในตัวของผู้หญิง INFJ ที่มักขาดความมั่นใจ สื่อสารไม่เก่ง เข้าใจยาก หรือมักดึงดูดคนที่มีปัญหาเข้ามาในชีวิต เนื้อหาจะเน้นการใช้จุดเด่นจากกลไกการทำงานของ Cognitive Function ประยุกต์เข้ากับทฤษฎีศิลปะการดึงดูดใจ (Art of Seduction) เพื่อสร้างกลยุทธ์เสน่ห์แบบ Passive Attractive ที่ช่วยให้ผู้หญิง INFJ สามารถนำเสนอตัวเองได้อย่างเป็นธรรมชาติ เป็นผู้รับอย่างภาคภูมิใจ และสามารถดึงดูดรวมถึงครองใจคนที่รักได้โดยไม่ต้องฝืนเป็นฝ่ายรุก

**Audience**
กลุ่มเป้าหมายหลักคือ ผู้หญิงที่มีบุคลิกภาพแบบ INFJ 

**Objective**
1. เพื่อให้รู้กลไกการสร้างเสน่ห์ของผู้หญิง INFJ แบบไม่ต้องเป็นฝ่ายรุก
2. เพื่อให้รู้จักกลไกการทำงาน Cognitive Function ของตัวเอง
3. เพื่อให้สามารถดึงดูดคนที่รักได้
4. เพื่อสร้างเสน่ห์ให้ในแบบของผู้หญิง INFJ
5. เพื่อสร้างผู้หญิง INFJ ให้มีกลยุทธ์ในการสร้างเสน่ห์และครองใจคนรักได้
6. สอนการใช้ Cognitive Function ของ INFJ เพื่อนำมาเป็นจุดเด่นในการสร้างเสน่ห์

**Problem**
1. ผู้หญิง INFJ ไม่มีความมั่นใจเพราะมักจะรู้สึกว่าตัวเองแปลก
2. ผู้หญิง INFJ มักดึงดูดแต่คนที่มีปัญหาเข้ามาในชีวิต
3. ผู้หญิง INFJ เป็นคนพูดน้อย จึงมักพลาดโอกาสในการนำเสนอ (Present) ตัวเอง
4. ผู้หญิง INFJ มักสื่อสารไม่เก่ง ชอบพูดอ้อมและไม่ตรงประเด็น
5. ผู้หญิง INFJ มักถูกมองว่าเข้าใจยากและดูลึกลับ
6. ผู้หญิง INFJ มักเป็นฝ่ายให้ โดยมีความคิดว่าการเป็นฝ่ายรับนั้นเป็นความผิด
